# Sistema de Controle de Acesso RFID - Hospital

Este projeto tem como objetivo desenvolver um sistema de controle de acesso para ambientes hospitalares, utilizando tecnologia RFID integrada a um sistema web e hardware com Arduino.

## Estrutura do Projeto
- **Frontend**: Interface do usuário feita em React.
- **Backend**: API construída com FastAPI (Python) para validação dos acessos.
- **Arduino**: Código C++ para leitura RFID e comunicação com o backend.
- **Docs**: Documentos de apoio como fluxogramas, banco de dados e design de telas.

## Tecnologias Utilizadas
- React (Vite) + Electron
- FastAPI (Python)
- PostgreSQL
- Arduino C++
- Docker
