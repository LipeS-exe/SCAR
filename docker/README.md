# O que o Docker precisa fazer no seu projeto:
#### Função do Docker:
Docker serve para padronizar o ambiente de desenvolvimento e execução, garantindo que o projeto funcione igual na máquina de todos — sem na minha máquina funciona.

#### O que o Docker deve fazer no seu projeto:
- Subir o back-end com FastAPI (e possivelmente com Uvicorn).

- Conectar ao banco de dados (PostgreSQL ou outro).

- Criar containers separados para cada serviço, se necessário.

- Facilitar o deploy para ambientes de produção ou teste.
