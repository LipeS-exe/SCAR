# Banco de Dados - SCAR
### Banco de dados — Estrutura e persistência
#### O que faz:

- Armazena dados de usuários, permissões, dispositivos RFID, logs de acesso.

- Permite consultas rápidas e confiáveis.

- Mantém integridade e segurança das informações.

#### Como fazer:

- Modelar tabelas: Usuários, Cartões RFID, Permissões, Logs de Acesso.

- Criar scripts SQL para criar e popular tabelas.

- Configurar conexão do back-end ao banco.

- Criar backups e documentação do modelo de dados.


Este diretório contém os arquivos relacionados ao banco de dados PostgreSQL do projeto.
