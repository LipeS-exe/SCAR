# Código relacionado a esta parte do sistema.
### Front-end — Interface do usuário
#### O que faz:

- Cria as telas para o usuário interagir (login, cadastro, painel de controle, notificações de acesso liberado/negado).

- Consome a API do back-end para mostrar e enviar dados.

- Valida campos (ex: formato do cartão RFID, usuário ativo).

- Exibe mensagens de erro e sucesso.

#### Como fazer:

- Usar framework JS moderno (React, Vue, etc).

- Comunicação via HTTP (fetch, axios) com FastAPI.

- Garantir responsividade e acessibilidade.

- Usar arquivos e componentes organizados (ex: /src/components/Login.js)
